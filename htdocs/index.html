Persistent ID Domain

This is a subdomain name used by SURROUND Australia Pty. Ltd. as a persistent namespace for a number of data and definitional assets, such as ontologies and vocabulary terms. The persistent identifiers allocated are linked to below.

Some of SURROUND's technical tools are listed within this domain too, see the Tools Page.

If you don't know why you're seeing this page, it may be that you clicked on a link that didn't work properly and you've ended up here at this page as a fallback option, rather than the specific item's home.

If you need assistance with any resources here, please Contact SURROUND!

Persistent Identifiers

  1. https://data.surroundaustralia.com/def/exem
    • Exemplification Ontology
    • Used to describe examples of defined things (classes, properties etc.) in other ontologies
    • Ontology in RDF: https://data.surroundaustralia.com/def/exem.ttl
  2. https://data.surroundaustralia.com/def/prof-roles
    • PROF Resource Roles Vocabulary
    • An extension to the vocabulary of Resource Roles publised within the [W3C](https://www.w3.org)'s [Profiles Vocabulary (PROF)](https://www.w3.org/TR/dx-prof/)
    • Vocabulary in RDF: https://data.surroundaustralia.com/def/prof-roles.ttl
  3. https://data.surroundaustralia.com/def/provworkflow
    • Prov Workflow Profile
    • A profile of the W3C's Provenace Ontology, PROV-O, used to describe the results of workflow execution
    • Profile in RDF: https://data.surroundaustralia.com/def/provworkflow.ttl
  4. https://data.surroundaustralia.com/def/uc
    • Use Case Ontology
    • An OWL representation of UML's Use Case model
    • Ontology in RDF: https://data.surroundaustralia.com/def/uc.ttl

PID System

The system used to maintain these PIDs is a very simple Apache web server with mod_redirect doing all the web link redirection. It is very similar in setup to the Australian Government Linked Data Working Group's PID Server, used to allocation PIDs for Australian government data and also the w3id.org systems mada available freely for PIDs.
